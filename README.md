# TASK 01 – Job Analysis & Description

## Project Title
Job Description for Mid-Level Data Analyst

---

# Job Title
Mid-Level Data Analyst

## Job Type
Full-Time

## Location
Remote / Hybrid

## Department
Data Analytics

---

# Job Summary
We are looking for a skilled and detail-oriented Mid-Level Data Analyst to join our growing team. The candidate will be responsible for collecting, analyzing, and interpreting data to help businesses make informed decisions. The ideal applicant should have strong analytical abilities, technical expertise, and problem-solving skills.

---

# Key Responsibilities
- Collect, clean, and analyze large datasets
- Prepare reports and dashboards for business insights
- Identify trends and patterns in data
- Work with teams to improve business performance
- Create data visualizations and presentations
- Monitor data accuracy and quality
- Support decision-making using analytical findings

---

# Required Technical Skills
- Python Programming
- SQL Database Management
- Microsoft Excel
- Data Visualization Tools (Power BI / Tableau)
- Data Cleaning and Processing
- Statistical Analysis
- Machine Learning Basics

---

# Soft Skills
- Communication Skills
- Problem-Solving Ability
- Team Collaboration
- Time Management
- Critical Thinking
- Attention to Detail

---

# Educational Requirements
- Bachelor’s degree in Computer Science, Data Science, Information Technology, or related field

---

# Experience Required
- 2–4 years of experience in data analysis or related roles

---

# Salary Range
₹5,00,000 – ₹9,00,000 per year (depending on skills and experience)

---

# Employee Benefits & Perks
- Flexible working hours
- Remote work opportunities
- Paid leave and holidays
- Professional development programs
- Performance bonuses
- Health and wellness support

---

# Preferred Qualifications
- Experience with real-world data projects
- Knowledge of cloud platforms
- Understanding of business intelligence tools

---

# Conclusion
The Mid-Level Data Analyst role is essential for organizations that rely on data-driven decision-making. This position requires a combination of technical expertise, analytical thinking, and communication skills to help businesses achieve better performance and growth.
